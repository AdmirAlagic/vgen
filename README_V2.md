## 🐛 Troubleshooting

### Video length still wrong?
**Check console output:**
```
Setting up animation: 2400 frames at 60 FPS = 40.00 seconds
```
The last number should match your audio duration.

### Camera still jerky?
**Verify you're running v2.0:**
```bash
# Check blender_generator.py line ~400
# Should see: keyframe_interval = 15
```

### Low quality output?
**Check settings:**
- Engine: Cycles (not EEVEE)
- Samples: 128+ (not 32)
- Denoising: Enabled

### Render taking too long?
**Try faster settings:**
- Reduce FPS: 60 → 30
- Reduce samples: 128 → 64
- Use EEVEE engine
- Lower resolution: 1080p → 720p
