## 🚀 How to Use

1. **Launch the App**: Run `python3 src/main.py`
2. **Select Audio File**: Choose a 10-15 second audio file for testing
3. **Configure Settings**:
   - Engine: Cycles (High Quality) or Eevee (Fast)
   - Samples: 64-128 for quality
   - FPS: 30-60
4. **Make sure Fast Mode is DISABLED** (unchecked)
5. **Click "🎬 Generate Video"**
6. **Watch the console** for the professional animation system output!
