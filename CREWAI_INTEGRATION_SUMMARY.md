## 🚀 Usage Commands

### Quick Start
```bash
# Setup CrewAI environment
python setup_crewai.py

# Run autonomous video generation
python autonomous_development.py --video audio.wav

# Run continuous improvement
python autonomous_development.py --continuous

# Use CrewAI crew commands
./crew "Optimize audio analysis for better quality"
```

### Advanced Usage
```bash
# Target broadcast quality
python autonomous_development.py --video audio.wav --quality broadcast

# Run specific crew task
python crewai_crew.py --crew "Optimize for 4K quality"

# Generate development report
python crewai_crew.py --report

# Self-improvement cycle only
python crewai_crew.py --self-improvement
```
