How to Generate Your Video:

  1. Upload your audio file above and configure visualization settings
  2. Click "Download Settings" to get the configuration file
  3. Install Python dependencies on your Mac:
    pip3 install -r requirements.txt
  4. Place your audio file in the same folder as the Python script
  5. Run the generator:
    python3 generate_video.py
  6. Get your professional MP4 video! Ready for YouTube, social media, etc.

💡 The Python script uses your Mac's FFmpeg for professional-quality encoding with perfect audio sync!
