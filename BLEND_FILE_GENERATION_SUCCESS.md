## 🎬 How to Use

1. **Generate blend files**:
   ```python
   from animator import create_mutating_cube_animation
   from audio_analyzer import EnhancedAudioAnalyzer
   
   # Analyze audio
   analyzer = EnhancedAudioAnalyzer("your_audio.mp3")
   features = analyzer.analyze_for_mutating_cube()
   
   # Generate blend file
   blend_file = create_mutating_cube_animation(
       audio_features=features,
       output_path="output/my_animation",
       quality_level="high",
       generate_blend=True
   )
   ```

2. **Open in Blender**: The generated `.blend` files can be opened directly in Blender to view and render the animations.

3. **Quality Levels**: Choose from `preview`, `fast`, `medium`, `high` based on your needs.
