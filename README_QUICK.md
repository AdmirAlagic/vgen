# VGenerator - Quick Start

## 🚀 **Fully Automated Audio Visualizer**

**Upload audio → Get professional video automatically!**

---

### ⚡ **2-Step Setup:**

```bash
# 1. One-time setup (installs FFmpeg + Python dependencies)
./setup.sh

# 2. Start the server  
./start.sh
```

### 🎵 **Generate Videos:**

1. **Open**: http://localhost:5000
2. **Upload audio file** (drag & drop)
3. **Choose settings** (visualization type, colors, quality)
4. **Click "Generate"** and wait 2-5 minutes
5. **Download your professional MP4 video!**

**That's it! Everything happens automatically.**

### 🛠️ **Troubleshooting:**

**"FFmpeg not found"**: `brew install ffmpeg`
**"Dependencies missing"**: `./setup.sh`
**Server won't start**: Make sure Python 3.7+ is installed
